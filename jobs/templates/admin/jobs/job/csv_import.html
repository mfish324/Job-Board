CSV Format Requirements

Your CSV file must include a header row with the following columns:

  • title (required) - Job title
  • company (required) - Company name
  • description (required) - Job description
  • location (required) - Job location
  • salary (optional) - Salary range or amount
  • is_active (optional) - true/false (defaults to true)

Example CSV:

title,company,description,location,salary,is_active
Software Engineer,Acme Corp,"Develop web applications using Python and Django",Chicago IL,$80000-$120000,true
Marketing Manager,Tech Inc,"Lead marketing campaigns and brand strategy",Remote,$70000-$90000,true
